                                                 第九章     异常
第57条：只针对异常的情况才使用异常。

第58条：对可恢复的情况使用受检异常，对编程错误使用运行时异常。

第59条：避免不必要地使用受检的异常。      

第60条：优先使用标准的异常。

第61条：抛出与抽象相对应的异常。

第62条：每个抛出的异常都要有文档。

第63条：在细节信息中包含能捕获失败的信息。

第64条：努力使失败保持原子性。

第65条：不要忽略异常。
